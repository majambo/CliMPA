# CliMPA
This repository contains the climate data analysis workflows, datasets, and derived products developed under the CliMPA project (Western Indian Ocean Marine Protected Area Climate Futures).
CliMPA aims to mainstream climate change data and information into Marine Protected Area (MPA) design, planning, and management in the Western Indian Ocean (WIO), with Mafia Island Marine Park (MIMP), Tanzania, serving as the pilot site.

This repository uses best-available open and FAIR data (no proprietary datasets), consistent with CliMPA principles including dowscaled CMIP6 data from Bio-ORACLE which offers essential physical, chemical, biological and topographic data layers.
